## 1. 신규 계정 생성 방법  🔴 (가장 중요)
- 현황: 앱에 자가 회원가입 페이지가 없음. 로그인(사용자 이름 + 비밀번호)만 존재.
- 질문: 외국인·신규 사용자가 계정을 어떻게 받나요? (초대 / 관리자 생성 / 가입 페이지 예정 등)
- 영향: 온보딩의 '입구'. 이게 없으면 챗봇이 잘 안내해도 결국 가입에서 막힘.
- 반영 위치: KB "로그인 / 계정" 문단.

## 2. "5분 무료"의 실제 동작
- 현황: 메인에 "5분 무료로 시작" 문구가 있으나, 무료 한도 로직이 코드에 없음.
- 질문: 무료 한도가 실제로 적용되나요? 어떻게? 한도 초과 시 어떻게 되나요?
- 반영 위치: KB "요금 / 무료 사용" 문단.

## 3. 생성 음성의 상업적 이용 정책
- 현황: 약관에 "생성 결과물의 이용 범위는 회사 정책 또는 별도 계약에 따름"으로만 되어 있어 모호.
- 질문: 사용자가 만든 음성을 광고·유튜브 등 상업적으로 써도 되나요? 조건은?
- 반영 위치: KB "생성한 음성의 사용 범위" 문단.
